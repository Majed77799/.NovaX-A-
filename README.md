# NovaX Monorepo

- Apps: `apps/mobile` (Expo SDK 53), `apps/web` (Next.js PWA)
- Services: `services/api` (Express TS)
- Packages: `packages/core`, `packages/ui`

## Commands
- dev: `npm run dev` (all)
- mobile: `npm run dev:mobile`
- web: `npm run dev:web`
- api: `npm run dev:api`
- typecheck: `npm run typecheck`
- lint: `npm run lint`
- clean: `npm run clean`
